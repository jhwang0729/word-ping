# Word Ping - App Specification

## 1. Overview

- **App Name**: Word Ping
- **Platform**: Cross-platform
- **Description**: 단어 사전 앱. 단어를 저장하고, 저장된 단어에 플래시 카드 기능을 제공하고 퀴즈를 풀 수 있으며, 알람 해제 시 퀴즈를 맞춰야 하는 기능이 핵심.
- **Target Users**: 영어 공부를 하고 싶은 모든 한국인들 혹은 외국인들
- **Tech Stack**: Flutter를 생각중이야

## 3. Features

### 홈
* 앱에 들어왔을때 가장 기본적인 화면이야
* 우선 모든 기능 페이지와 홈에 공통적으로 하단에 nav bar가 있을거야
  * 총 세가지 버튼이 있고
  * 왼쪽 : 단어장
  * 오른쪽 : 알람
  * 가운데 : 돋보기 모양을 하고 있고, 해당버튼을 누르면 홈 페이지로 항상 돌아와
    * 해당 버튼은 동그랗게 되어 있고, 동그라미 안에 돋보기 모양이 있어
* 화면 상단 (검색창이 최상단 보다는 조금 아래에 위치해 있어. 한 1/5?? 정도에 위치해 있어)
  * 유저가 검색할 수 있는 검색창이 있어
  * 검색창 왼쪽에는 돋보기 모양이 있어
  * 창은 왼쪽 오른쪽에 조금씩 margin이 있어
  * optional, 있으면 좋을것 같은 피쳐이지만, 구현이 가능한지 모르겠는 부분 : 유저가 단어를 type하기 시작하면 자동 완성을 해줬으면 좋겠어. 애플 키보드가 지원하는 것 처럼. 근데 지원하는 지 모르겠어, 한 번 알아봐줘. 자동완성 퀄리티가 떨어지면 그냥 없앨거야.
  * 오른쪽 상단에는 설정으로 누를 수 있는 버튼이 있어

### 검색
* 유저는 홈에서 검색을 시작할 수 있어.
  * 검색창은 위에서 설명한 대로 생겼고, 검색의 시작점이야
* 유저는 여기서 정확한 단어를 입력해야 할거고, 입력한 단어를 api를 통해서 단어의 뜻을 불러올거야.
* 우리는 다음과 같은 api를 고려하고 있어
  * FreeDictionaryAPI.com
  * 나중에 구현하기 전에 해당 홈페이지에 들어가서 document를 읽어봐
    * 추가로 Free Dictionary API 혹은 Wiktionary API (공식)도 생각중이니 한 번 찾아는 봐줘
* 유저가 해당 단어를 검색하면 바로 단어의 뜻을 설명하는 상세 페이지가 나올거야.
  * 해당 상세 페이지에서는 다음과 같은 기능을 지원해
    * 왼쪽 상단에 검색한 단어가 있고, 바로 옆에 해당 단어의 발음 기호를 표시할거야
      * 만약 api 답변으로 여러개가 여러개를 파싱해서 똑같은 발음 기호 갯수가 많은 것을 사용할거야
    * 오른쪽 상단에는 별표 모양이 있고, 해당 별표 모양을 클릭함으로써 단어장에 저장할 수 있어
      * 이 때, 단어장 섹션을 여러개 설정할 수 있고, 해당 페이지에서 저장하고 싶은 단어의 섹션을 설정하거나 단어의 섹션을 추가로 설정할 수 있어
    * 단어의 바로 밑에는 발음 기호를 바탕으로 미국 영어의 발음을 나타내는 기호를 만들어줘
    * 그 다음에는 단어의 뜻과 예문이 나올거야. 이 때, api의 답변이 여러개가 온다면 해당 리스트를 모두 파싱해서 보여줘. 너무 많다면 5개까지만 사용해줘

### 단어장
* 단어장은 비교적 간단해
* 해당 페이지로 들어가면 단어장의 여러 종류가 나올것이고, 해당 종류별로 오른쪽에 플래시 카드 혹은 퀴즈 tab을 만들거야
  * 이 때, 아무 단어도 저장이 되어 있지 않다면 tab을 비활성화 해줘
* 해당 페이지에서 단어장 종류를 삭제/추가할 수 있어
* tab에서 단어장 종류 이름과 더불어 저장되어 있는 단어 개수를 표시해줘
#### 플래시 카드
* 특정 단어장 종류에 있는 단어들을 섞어서 플래시 카드를 만들어 줘. 이 때, 두 가지 방법이 있고, 해당 방법은 설정에서 설정할 수 있어
  * 단어를 앞면으로 표현하는 경우
    * 뒤집으면 뜻이 나올거야. 근데 해당 단어의 뜻을 두개까지 표현해줘. 예문도 같이 표현해주면 좋을것 같아
  * 뜻을 앞면으로 표현하는 경우
    * 저장된 단어의 첫번째 뜻을 표현해줘

#### 퀴즈
단어장 페이지에 있는 단어장 종류별로 퀴즈 버튼을 클릭해서 해당 단어들에 대한 퀴즈를 볼 수 있어. 퀴즈는 해당 단어장 내에 있는 단어 중 최대 20개의 단어를 무작위로 선택해서 하나씩 단어를 보여주고 4개의 객관식 선택지를 주어 해당 단어의 뜻을 유저에게 맞추게 해. 4개중 1개의 뜻만 정답이야. 선택지를 클릭하면 해당 뜻을 선택한 것으로 인지하고 채점에 들어가. 만약 정답일 경우 초록색 ui로 해당 선택지가 정답임을 강조하고, 틀리면 붉은색 ui로 해당 선택지가 오답임을 강조한 후 다음 문제로 넘어가. 선택된 최대 20개의 단어들에 대해서 해당 작업을 반복하여 마지막 문제를 풀면 총 몇개를 맞췄는 지, 그리고 틀린 단어들의 뜻과 풀이를 보여준 후 확인을 누르면 다시 단어장 리스트 페이지로 돌아가. 그러면 퀴즈를 활성화하기 위한 저장된 최소 단어의 개수는 5개이고, 퀴즈 개수는 설정에서 설정할 수 있도록 하자 5, 10, 15, 20

### 알람
알람 페이지에 진입하면 알람 리스트를 보여줘. CRUD를 모두 할 수 있어. + 버튼을 통해서 신규 알람을 만들 수 있어. 알람마다 최대 12글자의 알람이름을 설정할 수 있으며 알람 시간 및 울리는 요일을 선택할 수 있어., 알람에 사용될 음악 및 사운드를 선택할 수 있어. 알람마다 단어장을 선택하고, 단어 알람의 활성화 여부를 on/off할 수 있어.  만약 활성화 되었다면 알람이 울릴때 해당 알람에 연결되어있는 단어장에서 무작위로 하나의 단어 풀이가 나오고 4개의 선택지중에서 맞는 단어를 맞춰야지만 알람이 꺼져. 한 번 틀릴때마다 맞춰야 하는 단어의 수가 한개씩 늘어나. 기본으로 맞춰야하는 단어의 개수를 알람마다 설정할 수 있어.
알람 리스트 페이지에서 토글을 통해서 해당 알람 사용여부를 결정할 수 있으며 내일 울리는 경우 색상이 강조돼. 그리고 어떤 어떤 요일에 울리는 알람인지도 알 수 있어. 리스트 페이지에서 이미 존재하는 알람을 클릭하면 수정할 수 있으며 맨 마지막 버튼으로 삭제도 할 수 있어.
새로운 알람을 만드는 페이지와 기존 알람을 선택해서 수정하는 페이지는 사실상 동일해.
